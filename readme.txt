== Description ==
My Todo List is a simple WordPress plugin that allows you to manage and display a todo list on your website. It leverages custom post types and custom fields to help you organize and manage tasks easily from the WordPress dashboard.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/my-todo-list` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcodes provided to display your todo list on any post or page.

== Usage ==
1. **Shortcode**: You can use the `[my_todo_list]` shortcode to display the todo list on any page or post.
2. **Custom Post Type**: The plugin creates a custom post type called "Todo" where you can add your tasks.
3. **Custom Fields**: Each todo item includes custom fields for setting additional details like due dates or priority.

== Frequently Asked Questions ==

= How do I display the todo list on my site? =
Simply use the `[my_todo_list]` shortcode on any page or post where you want the list to appear.

= Can I customize the appearance of the todo list? =
Yes, you can customize the appearance by overriding the CSS styles in your theme's stylesheet or by adding custom styles.

= Where can I add tasks? =
Tasks can be added through the "Todos" menu item in the WordPress dashboard after the plugin is activated.
